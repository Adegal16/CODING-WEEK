#### 1. Fixed Data Leakage in Data Processing
I identified that SMOTE was being applied before the train/test split, which causes 
data leakage — synthetic patients in the test set were based on training data. 

Before (wrong):
```python
# SMOTE on entire dataset then split — data leakage!
df = handle_imbalance(df)
X_train, X_test, y_train, y_test = split_data(df)
```

After (my fix):
```python
# Split first, then SMOTE on training only — no leakage
X_train, X_test, y_train, y_test = split_data(df)
X_train, y_train = apply_smote(X_train, y_train)
```

#### 2. Built the Training Pipeline (src/train_model.py)

**Scaling features** — puts all features on the same scale so no feature dominates:
```python
scaler = StandardScaler()
X_train_scaled = scaler.fit_transform(X_train)  # learn + apply
X_test_scaled = scaler.transform(X_test)         # apply only
```

**Training 4 models using a loop** — cleaner than writing each one separately:
```python
models = {
    "Logistic Regression": LogisticRegression(random_state=42, max_iter=1000),
    "Random Forest": RandomForestClassifier(n_estimators=100, random_state=42),
    "XGBoost": XGBClassifier(n_estimators=100, random_state=42, eval_metric='logloss'),
    "LightGBM": LGBMClassifier(n_estimators=100, random_state=42, verbose=-1)
}

for name, model in models.items():
    model.fit(X_train_scaled, y_train)
    y_pred = model.predict(X_test_scaled)
```

**Evaluating with 5 metrics** — accuracy alone is misleading with imbalanced data:
```python
acc  = accuracy_score(y_test, y_pred)
prec = precision_score(y_test, y_pred)
rec  = recall_score(y_test, y_pred)
f1   = f1_score(y_test, y_pred)
auc  = roc_auc_score(y_test, y_pred)
```

**Saving the best model** — so the Streamlit app can load it without retraining:
```python
joblib.dump(best_model, "models/best_model.pkl")
joblib.dump(scaler, "models/scaler.pkl")
```

In medicine, recall is critical — missing a dying patient is worse than a false alarm.
The confusion matrix showed only 3 missed deaths out of 41.

#### 4. Created Model Evaluation Notebook
Built `notebooks/model_evaluation.ipynb` with:

### Key Findings

- **Time** (follow-up period) is the strongest predictor of death
- **Serum creatinine** (kidney function) is the second most important
- **Ejection fraction** (heart pumping efficiency) is the third
- Features like smoking, diabetes, and sex had minimal impact
- These findings align with medical literature
